TLDR · denylist

One host per line. TLDR stays off on these hosts and their subdomains (e.g. localhost, google.com). The Chrome Web Store is always off and can't be removed.

Hover previews

Show a quick popup with the leading note for a link when you hover it, on any page. Off by default. Turning this on asks Chrome to grant TLDR access to read the pages you visit, so it can detect which link you're hovering — turning it back off removes that access again.
